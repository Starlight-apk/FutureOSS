## 安全改进

### 1. CORS 安全
- 不再允许所有来源的跨域请求

### 2. 网络安全
- 默认绑定本地接口，避免暴露到所有网络
- API 认证支持（空API_KEY时自动禁用）

### 3. 日志安全
- 支持结构化日志（JSON格式）
- 文件日志支持，避免敏感信息输出到控制台
- 日志轮转，防止日志文件无限增长
